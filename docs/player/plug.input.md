# plug.input

[example](./plug.input.ts)

## events
- `playerInput:jumpStart`  
プレイヤーがジャンプを開始したときに発火  
- `playerInput:jumpEnd`  
プレイヤーがジャンプを終了したときに発火
- `playerInput:sprintStart`  
プレイヤーが走り始めしたときに発火  
- `playerInput:sprintEnd`  
プレイヤーが走り終わったときに発火  
- `playerInput:sneakStart`  
プレイヤーがしゃがみ始めしたときに発火  
- `playerInput:sneakEnd`  
プレイヤーがしゃがみ終わったときに発火
